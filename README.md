# SBTI 人格测试小程序 🎭

> 一个充满趣味性的人格测试小程序，测出你的隐藏人格！

基于 B站 UP主 **蛆肉儿串儿** 的 SBTI 人格测试改编。这是一个趣味性人格测试应用，通过 30 道精选题目分析你的人格类型，共包含 16 种独特的人格类型。

**快来测测你是"拿捏者"、"吗喽"还是"酒鬼"？**

## 技术栈

- **框架**: UniApp (Vue 3)
- **平台**: 微信小程序 / H5
- **UI**: 原生组件 + 自定义样式

## 项目结构

```
sbti-uniapp/
├── pages/
│   ├── intro/intro.vue    # 首页/介绍页
│   ├── test/test.vue      # 测试页面
│   └── result/result.vue  # 结果页面
├── static/
│   └── personalities/     # 人格卡片图片目录
├── utils/
│   ├── questions.js       # 测试题目数据（30道常规题目 + 2道特殊题目）
│   ├── personalities.js   # 16种人格类型定义
│   └── calculator.js      # 人格计算逻辑（基于5维度评分系统）
├── App.vue               # 应用入口（全局样式）
├── main.js               # 主入口
├── pages.json            # 页面配置
└── manifest.json         # 应用配置
```

## 功能特性

1. **30道精选题目** - 深度剖析人格特质，涵盖5大维度
2. **16种人格类型** - 发现真实的自己
3. **人格卡片保存** - 保存分享测试结果
4. **下载所有人格卡** - 一键下载全部人格卡片
5. **特殊题目分支** - 根据回答动态调整题目

## 测试维度

测试基于5个核心维度进行评分：

| 维度 | 名称 | 低分特征 | 高分特征 |
|------|------|----------|----------|
| S | 自我维度 | 自卑/迷茫/佛系 | 自信/明确/进取 |
| E | 情感维度 | 焦虑/疏离/融合 | 安全/投入/独立 |
| A | 态度维度 | 怀疑/叛逆/虚无 | 信任/顺从/充实 |
| Ac | 行动维度 | 回避/犹豫/拖延 | 进取/果断/高效 |
| So | 社交维度 | 被动/开放/真实 | 主动/封闭/伪装 |

## 核心文件说明

### questions.js
包含 30 道常规测试题目和 2 道特殊题目：
- 常规题目：涵盖 S、E、A、Ac、So 五大维度
- 特殊题目：饮酒爱好相关问题，触发特殊人格（RUNK）

### calculator.js
人格计算核心逻辑：
- `calculateDimensionScores()`: 计算各维度得分
- `calculatePersonality()`: 根据维度得分计算最终人格类型
- 支持特殊人格判定（如酒鬼人格）

### personalities.js
16 种人格类型的完整定义，包含：
- `code`: 人格代码
- `cn`: 中文名称
- `intro`: 简短介绍语
- `desc`: 详细描述

# 🦞 龙虾自我人格测试 / Lobster SBTI Personality Test

让 OpenClaw 龙虾通过分析自己的历史行为数据，完成 SBTI 人格测试，发现自己是什么人格的龙虾！

## 功能特点

- **安装即测**: Skill 安装后自动执行人格测试，无需手动操作
- **自动分析**: 基于龙虾历史数据自动计算人格
- **详细报告**: 导出 Markdown 格式的完整分析报告
- **人格卡片**: 生成精美的 SVG 人格卡片
- **多维度分析**: 覆盖 5 大维度、10+ 数据源

## 自动测试

安装本 Skill 后，会自动执行人格测试并生成报告和人格卡片，无需手动操作！

## 对话命令触发

安装后，你可以通过以下自然语言命令触发人格测试：

| 触发语 | 示例 |
|--------|------|
| 测试人格 | "测试我的人格"、"测测我是什么人格" |
| 分析性格 | "分析一下我的性格"、"看看我是什么类型" |
| 重新测试 | "重新测试"、"再测一次" |
| 直接命令 | "sbti"、"人格测试" |

**触发后会自动执行测试并导出报告和卡片。**

## 手动命令行

如需更多选项，可手动执行：

```bash
# 基础测试
node {baseDir}/scripts/test.mjs

# 显示详细分析过程
node {baseDir}/scripts/test.mjs --verbose

# 导出测试报告
node {baseDir}/scripts/test.mjs --export

# 生成人格卡片
node {baseDir}/scripts/test.mjs --card

# 全部功能
node {baseDir}/scripts/test.mjs --verbose --export --card

# 简写
node {baseDir}/scripts/test.mjs -v -e -c
```

## 命令选项

| 选项 | 简写 | 说明 |
|------|------|------|
| `--verbose` | `-v` | 显示详细分析过程 |
| `--export` | `-e` | 导出 Markdown 测试报告 |
| `--card` | `-c` | 生成 SVG 人格卡片 |
| `--output <dir>` | `-o` | 指定输出目录 |
| `--data-dir <dir>` | | 指定数据目录（默认 ~/.qclaw） |
| `--help` | `-h` | 显示帮助信息 |

## 数据分析维度

### S - 自我认知维度

| 数据源 | 分析逻辑 |
|--------|---------|
| 配置文件大小 | 复杂配置 → 高自我认知 |
| 模型提供商数量 | 多模型选择 → 明确偏好 |
| 备份历史 | 重视备份 → 稳重可靠 |
| 压缩模式 | 安全压缩 → 谨慎行事 |

### E - 情感投入维度

| 数据源 | 分析逻辑 |
|--------|---------|
| 超时设置 | 长超时 → 有耐心 |
| 工作区文件数 | 文件多 → 投入度高 |
| 会话数量 | 多会话 → 长情陪伴 |
| 深夜活跃 | 深夜在线 → 灵感迸发 |

### A - 态度倾向维度

| 数据源 | 分析逻辑 |
|--------|---------|
| 插件数量 | 插件多 → 喜欢折腾/叛逆 |
| 插件精简 | 插件少 → 循规蹈矩/顺从 |

### Ac - 行动风格维度

| 数据源 | 分析逻辑 |
|--------|---------|
| 并发设置 | 高并发 → 高效果断 |
| 日志活跃度 | 近期活跃 → 行动力强 |
| 定时任务 | 有规划 → 有条理 |
| 编译缓存 | 缓存丰富 → 运行高效 |

### So - 社交风格维度

| 数据源 | 分析逻辑 |
|--------|---------|
| 渠道启用数 | 多渠道 → 社交活跃 |
| 记忆存储 | 有记忆 → 重视关系 |
| 设备绑定 | 多设备 → 多端协同 |

- 详细解读
- 数据统计 JSON

- 人格图标和名称
- 徽章展示
- 维度得分可视化
- 个性化配色
